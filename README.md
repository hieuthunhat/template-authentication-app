- Serving as a template for every simple client-server website application, it aims to reduce the need for redesigning the core security of the websites
# Flow
- Template here is a full-stack app using role-based authentication and authorization to manage users' actions

# Cores
- Role-based authorization
- JWT and cookie storage
- Basic Socket.io implementation for the notification and message features
